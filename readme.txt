Resizes uploaded images automaticly to "Large Size Max Width / Height"

== Description ==

The Auto Image Resizer does exactly what the name says. The plugin resizes all uploaded images automaticly to the maximal "Large Size" you set in your WordPress Admin Panel > Settins > Media.

This helps to save your web storage if you do not need the (original) images that are bigger than the "Large Size" images.

== Installation ==

Installation as usual.

1. Unzip and Upload all files to a sub directory in "/wp-content/plugins/".
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Upload an image to your WordPress System
1. You like that the image was resized to max size?
